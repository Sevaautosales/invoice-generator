# Seva Auto Sales

This project is separated into a frontend and a backend for better organization and scalability.

## Project Structure

- `frontend/`: Next.js application (User Interface)
- `backend/`: Express server (Server Side API)

## Getting Started

### Backend

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example` and add your Supabase credentials.
4. Start the server:
   ```bash
   npm run dev
   ```

### Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Development

- Frontend runs on: [http://localhost:3000](http://localhost:3000)
- Backend runs on: [http://localhost:5001](http://localhost:5001)
